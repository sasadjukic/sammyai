# Effective Conversations

SammyAI works best as a conversation. Instead of asking for a finished novel in one prompt, guide the system through ideas, constraints, drafts, critique, and revision.

---

## The Dialogue Mindset

Treat SammyAI like a collaborator.

* **Ask before drafting:** Start with questions such as "What would make this conflict sharper?"
* **Correct directionally:** If an answer is close, explain what should change instead of starting over.
* **Name the role:** Choose Brainstormer, Writer, Editor, Critic, or Assistant based on the task.
* **Use context deliberately:** Reference only the files and memories needed for the current step.

> [!TIP]
> Strong scenes often come from several focused turns instead of one large request.

---

## Prompting vs. Conversing

| One-Shot Request | Conversation |
| :--- | :--- |
| "Write a scene where two people argue about a car." | "Elias and Sarah argue about their old car, but the real subject is their failing marriage. What opening beat makes that subtext clear?" |
| Usually generic | Easier to steer toward a specific emotional target |

---

## Use Agents Intentionally

* **Brainstormer:** Generate options before you commit.
* **Writer:** Draft the selected direction.
* **Critic:** Identify problems without changing files.
* **Editor:** Propose reviewed file changes.
* **Assistant:** Ask general questions or inspect project context.

---

## Keep Sessions Focused

Use **New Chat** when the topic changes, the thread is long, or the model starts repeating itself. Summarize important outcomes and save durable facts as persistent memory when they should carry forward.

---

## Quick Tips

1. Ask follow-up questions.
2. Explain why a change is needed.
3. Reference the exact file when file content matters.
4. Use relative paths for ambiguous filenames.
5. Review any proposed change set before applying it.
