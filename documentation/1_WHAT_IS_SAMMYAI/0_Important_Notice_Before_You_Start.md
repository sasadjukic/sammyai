## **Welcome to the Alpha 0.3.1 Release of SammyAI!**

This release of SammyAI Alpha 0.3.1 brings significant architectural improvements, a refreshed user interface, and more granular control over your AI writing experience.

**Current Status:**

*   **Version:** SammyAI Alpha 0.3.1
*   **Development Stage:** Alpha
*   **User Interface:** Refreshed and redesigned. Key tools (RAG, Context Injection, Diff Edits) have been moved to the Chat Panel for better accessibility.
*   **Core Engine:** Improved system prompt logic and dynamic model management.
*   **Ongoing Development:** We are actively refining the creative writing engine and expanding provider support.

**Key Highlights of Alpha 0.3.1:**

*   **Dynamic LLM Setup:** You are no longer limited to hardcoded models. You can now dynamically configure up to 15 LLM models across local (Ollama) and cloud providers (OpenAI, Anthropic, Google).
*   **LLM Settings Presets:** Easily toggle between **Exploratory**, **Balanced**, and **Focused** modes to automatically tune temperature and top_p for your specific task.
*   **Improved System Prompt:** Introducing dynamic modes—**Ideation**, **Descriptive**, and **Dialogue**—designed to deliver clearer and more imaginative output.
*   **Reproducible Results:** A new `seed` parameter has been added to the LLM Settings panel for more consistent generation when needed.

**Important Notes:**

*   This is an Alpha version intended for early adopters. While core features are functional, expect regular updates as we stabilize new systems.
*   We highly value your feedback and bug reports during this transition to dynamic configurations!
