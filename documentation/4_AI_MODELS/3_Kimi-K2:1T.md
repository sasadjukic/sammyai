# Kimi K2:1T

Kimi K2:1T, developed by Moonshot AI, is a remarkable Mixture-of-Experts (MoE) model boasting an astounding 1 trillion total parameters. It is uniquely positioned as a cloud-based powerhouse that provides deep, human-like reasoning for the most demanding creative projects.

## Why Kimi K2:1T?

- **Colossal Scale:** At 1 trillion parameters, Kimi K2:1T offers exceptional nuance in story generation, character development, and intricate world-building.
- **Large Context Window:** Features a **256k context window**, making it ideal for managing long-form research and complex narrative structures.
- **Intelligent Reasoning:** Excels at maintaining consistency across large volumes of data, perfect for complex world-building.

## How to Get Your API Key

To access Kimi K2:1T in SammyAI, you must have an Ollama account and an API key:

1. **Sign Up / Sign In:** Go to [ollama.com](https://ollama.com/) and create an account or sign in to your existing one.
2. **Access Cloud Keys:** Navigate to your account settings or the cloud dashboard.
3. **Generate Key:** Follow the prompts to generate a new API key for cloud access.
4. **Configure SammyAI:** Copy your key and enter it in SammyAI (Settings > API Keys) under the Ollama provider.
