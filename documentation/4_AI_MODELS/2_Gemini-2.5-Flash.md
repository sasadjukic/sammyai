# Gemini 2.5 Flash

Gemini 2.5 Flash is Google's most versatile and efficient cloud-based model, engineered for high-speed performance and complex creative tasks. It offers an exceptional balance of speed, reasoning capability, and an industry-leading context window.

## Key Features for Writers

- **Massive Context Window:** With support for up to **1 million tokens**, Gemini 2.5 Flash can analyze entire novels, series-wide world-building documents, and vast research archives simultaneously.
- **Exceptional Speed:** Optimized for rapid response times, making it perfect for real-time brainstorming and fast-paced drafting.
- **Nuanced Reasoning:** Excels at maintaining narrative consistency and understanding complex character motivations over long spans of text.
- **Generous Free Tier:** Google provides a substantial daily free usage tier, making it one of the most cost-effective "heavyweight" models available.

## How to Get Your API Key

To use Gemini 2.5 Flash within SammyAI, you need a Google Gemini API Key. It takes only a few minutes to set up:

1. **Visit Google AI Studio:** Go to [Google AI Studio](https://aistudio.google.com/).
2. **Create a Project:** If you haven't already, sign in with your Google account and follow the prompts to create or select a project.
3. **Generate API Key:** Click on **"Get API key"** in the sidebar and then click **"Create API key in new project"**.
4. **Configure SammyAI:** Copy your key and paste it into SammyAI's API configuration menu (Settings > API Keys).
