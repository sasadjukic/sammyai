# Getting Started with SammyAI

To start using SammyAI, you can choose between two main installation methods depending on your preferences and technical setup.

### Installation Options

> [!IMPORTANT]
> Before you proceed, ensure you have reviewed the [Important Notice](file:///home/sasa/Desktop/Projects/sammyai_v1/documentation/1_WHAT_IS_SAMMYAI/0_Important_Notice_Before_You_Start.md) to understand the requirements and privacy considerations.

1. **Download from Source (GitHub)**
   This is the recommended method for users who want to stay up-to-date with the latest changes, customize their own experience, or perhaps contribute to the project. You can download the source code directly from the [SammyAI GitHub page](https://github.com/sasadjukic/sammyai_v1).

2. **Run via Docker**
   For a more streamlined, containerized experience that manages dependencies for you, SammyAI is available as a Docker image. This is ideal for quick testing and ensuring a consistent environment.
