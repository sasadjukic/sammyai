# Getting Started with SammyAI

SammyAI can be installed from source or run through Docker. Source installation is the most useful path for alpha testing because it gives you direct access to the newest code and project files.

## Installation Options

> [!IMPORTANT]
> Before you proceed, review the Important Notice in `documentation/1_WHAT_IS_SAMMYAI/0_Important_Notice_Before_You_Start.md` for release status and privacy considerations.

1. **Install from Source**
   Use this method if you want to test the latest changes, inspect the code, customize the app, or contribute fixes.

2. **Run with Docker**
   Use this method if you want a containerized environment. GUI applications in Docker may require extra display configuration, especially on Windows and macOS.

Detailed instructions:

* [Install from Source](2_From_Source.md)
* [Run with Docker](3_Docker.md)
